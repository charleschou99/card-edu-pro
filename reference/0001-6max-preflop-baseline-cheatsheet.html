Reference 0001 · Mission link: competition readiness through disciplined fundamentals

Position Order (6-max)

Action order preflop (first to act to last): UTG -> HJ -> CO -> BTN -> SB -> BB.

Core rule: the earlier you act, the tighter your starting range should be.

Beginner RFI Baseline (Raise First In)

Position Play style Simple baseline examples
UTG / HJ Tight Big pairs, strong broadways, strong suited aces (for example: AA-TT, AK, AQs, AJs, KQs)
CO Medium Add more pairs, more suited aces, and more broadways
BTN Wide Widest opening range; positional advantage allows profitable expansion
SB Mixed Open selectively, account for being out of position postflop

This is a starter baseline for training discipline, not a final solved strategy.

Pre-session Checklist (2 minutes)

  1. Today I prioritize position and fold discipline over action.
  2. I only play hands that fit my current position baseline.
  3. If unsure between call/fold, I compute rough pot odds.
  4. I end the session if stop-loss is reached.
